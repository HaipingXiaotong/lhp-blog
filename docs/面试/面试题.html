  16. promise.all实现
    function all(arr) {
    let length = arr.length
    let count = 0
    <!-- 创建出length个内存区 -->
    let arrs = new Array(length)
    return new Promise((resolve, reject) => {
        arr.forEach((item, index) => {
            item(index).then(function (res) {
                arrs[index] = res
                count++
                if (count === length) {
                    resolve(arrs)
                }
            }, function (err) {
                reject(err)
            })
        });
    })
}

    http缓存机制

强缓存和弱缓存,往返缓存（Back/Forward cache） 往返缓存（Back/Forward cache，下文中简称bfcache）是浏览器为了在用户页面间执行前进后退操作时拥有更加流畅体验的一种策略。该策略具体表现为，当用户前往新页面时，将当前页面的浏览器DOM状态保存到bfcache中；当用户点击后退按钮的时候，将页面直接从bfcache中加载，节省了网络请求的时间。

一道经典题型，100万个数据中，找出前10大数据，最快的方法是什么

  • 最先想到的是全部排序，最快的排序算法的时间复杂度一般为O（nlogn），如快速排序
  • 局部淘汰法，用一个容器保存前10个数，然后将剩余的所有数字----与容器内的最小数字相对比。如果所有后续的元素都比容器内的这10个数还要小，那么这10个数就是最大的数，如果某一后续元素比容器内的最小元素大，则把最小元素删除，并将该元素插入容器
